## Contributing

To run a web dev server you can execute `mix dev` on the project root folder.
Then the application will be server on `http://localhost:4001`

You can find the server configurations, options and routes on the [server.ex](./lib/bootstrap_live/server.ex) file!

## Installation

You can find this package on [Hex](https://hex.pm/editor_live), the package can be installed
by adding `editor_live` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:editor_live, "~> 0.1.0"}
  ]
end
```
